Dependencies
 numpy
 tkinter
 matplot
 scipy	

This is an Automatic Plotter if you select the folder version, it will run on all files in the 
folder and save the plots to a folder called plots. The file select will only run on the file
that you pick and then it will save the plot to the directory the script is in. 

This will plot the Wavelength and Voltage and plot the maxima. It will also give an average distance
between the maxima at the top of the graph.
